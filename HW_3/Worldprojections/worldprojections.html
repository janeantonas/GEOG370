In this project I learned how to display images in different projections:

Describe in your own words how you displayed the map in different projections using QGIS

In order to change the projection model used in each map display, I first clicked on the icon in the bottom right corner of the screen. From there, QGIS allows any projection model to be selected. Installing the Indicatrix Mapper plugin allowed me to show the Indicatrix circles on each map. These circles are helpful in identifying which aspect of the map is distorted.
